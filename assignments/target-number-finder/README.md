# 📘 Assignment: Algorithmic Calculator - Target Number Finder

## 🎯 Objective

Build a calculator-style Python program that finds two numbers whose sum matches a target value. In this assignment, you will practice algorithmic problem solving with loops, conditionals, and list processing.

## 📝 Tasks

### 🛠️ Build the Input and Search Logic

#### Description
Set up the program to read a list of numbers and a target number, then search for a valid pair by checking combinations of values.

#### Requirements
Completed program should:

- Accept a list of integers and a target integer.
- Use loops and conditionals to test possible pairs of numbers.
- Stop and report the first valid pair found that adds to the target.
- Print a clear message when no valid pair exists.


### 🛠️ Improve the Calculator Output

#### Description
Refine the program so the output is easy to understand and verify, including example calculations and index positions of matched numbers.

#### Requirements
Completed program should:

- Display the pair values and their index positions when a match is found.
- Show the calculation in the format a + b = target.
- Handle repeated values correctly (for example, target 10 from [5, 5, 3]).
- Include at least three test cases in the script, with at least one case where no pair exists.
